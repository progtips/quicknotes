# Favicon для QuickNotes

## Требования

Проект ожидает файл `favicon.png` в директории `assets/`.

## Как создать favicon

### Вариант 1: Использовать онлайн генератор
1. Перейдите на https://favicon.io/ или https://realfavicongenerator.net/
2. Загрузите изображение или создайте иконку
3. Скачайте `favicon.png` (размер 32x32 или 64x64 пикселей)
4. Сохраните файл как `packages/mobile/assets/favicon.png`

### Вариант 2: Создать простой favicon
1. Создайте изображение размером 32x32 или 64x64 пикселей
2. Используйте любой графический редактор (Paint, GIMP, Photoshop и т.д.)
3. Сохраните как PNG файл
4. Назовите файл `favicon.png`
5. Поместите в `packages/mobile/assets/`

### Вариант 3: Использовать временную заглушку
Можно использовать любой PNG файл как временную заглушку. Ошибка 500 для favicon не критична и не влияет на работу приложения.

## Текущая конфигурация

В `app.json` и `app.config.ts` указано:
```json
"web": {
  "favicon": "./assets/favicon.png"
}
```

## Проверка

После добавления файла:
1. Перезапустите Expo: `npx expo start --web --clear`
2. Ошибка `GET http://localhost:8081/favicon.ico 500` должна исчезнуть
